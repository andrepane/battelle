# Fuentes normativas Battelle

Las fuentes antiguas versionadas (PDF, Excel derivados, extracciones OCR, auditorías visuales y JSON normativos) fueron retiradas del árbol actual para preparar una incorporación controlada de nuevas fuentes estructuradas.

Categorías previstas:

- `percentiles/`: Excel manuales estructurados para conversiones PD→PC.
- `edades_equivalentes/`: Excel manuales estructurados para edades equivalentes.
- `conversiones_generales/`: Excel manuales estructurados para conversiones PC→z/T/CI/ECN u otras tablas generales autorizadas.
- `screening/`: Excel manuales estructurados del screening, mantenidos separados del Battelle completo.

Todavía no hay datos normativos activos en esta rama. Los Excel futuros serán la única fuente autorizada para regenerar baremos; no deben añadirse datos provisionales ni ejemplos inventados.
